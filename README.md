(Задание 2)
Der Morgen graut, der Wald erwacht,
Verflogen ist die tiefe Nacht.
Ein goldnes Licht fällt auf die Höhen,
Ein sanfter Wind beginnt zu wehen.

Der Nebel tanzt auf kühlem Grund,
Die Vögel tun ihr Liedchen kund.
Ein Bach fließt murmelnd durch das Tal,
Und glitzert wie ein Sonnenstrahl.

Ich wandre still den Pfad entlang,
Und lausche diesem Waldgesang.
Die Seele ruht, das Herz wird weit,
In dieser schönen Einsamkeit.

So steh ich hier in stummer Ruh,
Und schau dem Spiel des Lebens zu.
Ein Zauber liegt in diesem Sein,
Gehüllt in klaren Morgenschein.
